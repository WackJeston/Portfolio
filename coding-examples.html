Image Generator

This code uses an API called Picsum to access random images and create sectioned lists.

In a separate file, I used Axios and the math constructor to access the API and select a random image.
In this document, I then used the data to create multiple variables and display the random Image. Using if statements and an array of used emails, my function is able to work out if an email has been used before, and then either create a new section or add an image to an existing one.
The following function decides how large the section containing the list of images should be. It works this out depending on how many sections there are.

Portfolio Menu

This is the code I wrote to control the pop-up menu used in the mobile layout of this site.

I used CSS Keyframes to create animations for the menu appearing and disappearing.
By using a combination of if statements and click events I made three functions. One to control the menu opening and closing when the button is clicked. Another to close the menu if anything else is clicked. And the last function used the resize event to close the menu if the window grows too wide for mobiles.
All of these functions use the setTimeout function to wait for the keyframe animations to complete before moving on.

Form Validation

This is the code I wrote to handle the form validation and animation for this site.

I created individual functions to validate each of my inputs. Then I used a click event to check all my validations are correct. If they are all correct, a keyframe is run to allow the form to smoothly exit and be replaced by a thank you message.
I also created a function to control the size of the form zone depending on the window size, and whether the form has been completed or not. This is run when the page is loaded and every time the window is resized.
